CatByte needs an AI backend

CatByte is your gaming AI companion. Connect it to any of these backends:

  • Ollama — free, private, runs on your GPU
  • OpenClaw — uses your ChatGPT subscription
  • LM Studio — local models with visual UI
  • OpenAI API — GPT-4o with API key

Open Settings → CatByte to choose and configure your backend.

ROM Directories

Folders containing ROM files. Organize ROMs into system subdirectories (e.g., SNES/, GBA/, PS1/) for automatic detection.

Local Game Directories

Folders containing standalone .exe games or portable installs. Each subfolder becomes a game entry.

Library

Force a full rescan of all configured sources.

BIOS Files

Some systems need BIOS files to run. Add folders containing your BIOS files — YancoHub auto-detects them by filename and checksum.

ⓘ GBA and PS1 have free open-source BIOS alternatives bundled with YancoHub. Other systems need original BIOS files.

Visible Systems

Choose which retro systems appear in your library. Hidden systems are still scanned — just not shown.

Emulator Auto-Setup

YancoHub auto-downloads RetroArch + cores for your ROM library. No install needed.

External Emulators

For PS2, PS3, GameCube, Wii, Dreamcast, and Saturn — YancoHub can launch RetroArch or standalone emulators.

LaunchBox Artwork

Point to your LaunchBox install folder to use its artwork for game covers — no files are copied.

About

YancoHub is a unified game launcher that aggregates Steam, Epic, GOG, Xbox, EA, Ubisoft, Battle.net, local games, and retro ROMs into one cinematic interface.
