CatByte needs an AI backend

CatByte is your gaming AI companion. Connect it to any of these backends:

  • Ollama — free, private, runs on your GPU (recommended)
  • LM Studio — local models with visual UI
  • OpenAI API — GPT-4o and other models with API key
  • OpenClaw — OpenAI-compatible gateway with your own keys
  • Custom — any OpenAI-compatible endpoint

Open Settings → CatByte to choose and configure your backend.

BIOS Files

Some systems need BIOS files to run. Add folders containing your BIOS files — YancoHub auto-detects them by filename and checksum.

ⓘ GBA and PS1 have free open-source BIOS alternatives bundled with YancoHub. Other systems need original BIOS files.
