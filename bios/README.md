# YancoHub BIOS Files

## Shipped (Open Source)

Place these in this directory:

- `gba_bios.bin` — Open-source GBA BIOS (Cult-of-GBA, MIT license)
  Download: https://github.com/Cult-of-GBA/BIOS

- `scph1001.bin` — OpenBIOS for PS1 (MIT license)
  Download: https://pcsx-redux.consoledev.net/openbios/

## User-Provided

Place proprietary BIOS files in the `user/` subdirectory:

- `neogeo.zip` — Neo Geo BIOS (for Neo Geo arcade games)
- `bios7.bin`, `bios9.bin`, `firmware.bin` — Nintendo DS BIOS + firmware
